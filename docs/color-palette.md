# Color Palette - GoAndBe Traveler App

This color palette is derived directly from the "GoAndBe" brand logo to ensure a cohesive and professional visual identity across the application.

## Brand Colors

| Role | Color Name | Hex Code | Visual Description |
| :--- | :--- | :--- | :--- |
| **Primary** | Deep Navy | `#1A3254` | Extracted from the "GoAndBe" text. Used for App Bars and primary branding. |
| **Secondary** | Mountain Teal | `#5AB1B0` | Extracted from the top of the mountain icon. Used for FABs and active states. |
| **Accent** | Glacial Blue | `#367191` | Extracted from the mountain waves. Used for links, icons, and subtle highlights. |
| **Background** | Off-White | `#F5F7F9` | A cool-toned neutral for screen backgrounds to maintain a clean look. |
| **Surface** | Pure White | `#FFFFFF` | Used for cards, dialogs, and elevated UI components. |
| **Error** | Burnt Coral | `#E63946` | A complementary warm tone used for validation errors and delete actions. |

## Typography Colors

| Usage | Color Name | Hex Code |
| :--- | :--- | :--- |
| **Headings** | Deep Navy | `#1A3254` |
| **Body Text** | Dark Slate | `#2D3436` |
| **Secondary Text** | Dim Grey | `#636E72` |

## UI Implementation Strategy
* **Primary Identity:** The Deep Navy should be the dominant "grounding" color for navigation elements.
* **Call to Action:** The Mountain Teal should be reserved for the most important actions (e.g., "Create Trip") to ensure it draws the user's eye.
* **Readability:** Using Deep Navy for text against an Off-White background provides high contrast for travelers viewing their itinerary in various lighting conditions.
